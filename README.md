# User Guide

<ol>
<li>Find a favourable forex broker website (For this project, we used Tickmill Ltd.)
<p>Tickmill URL: <a href="https://secure.tickmill.com?utm_campaign=ib_link&amp;utm_content=IB52499319&amp;utm_medium=%2430+Welcome+Account&amp;utm_source=link&amp;lp=https%3A%2F%2Ftickmill.com%2Fpromotions%2Fwelcome-account%2F" target="_blank" rel="noopener">Tickmill (FSA Regulated)</a></p>
</li>
<li>Register a demo account (Or register for a real account, but this requires identity verification by uploading the identity documents to the broker)
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwAH2lswkN1w40-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Set up the principle(the "Virtual Deposit" field as circled below) of the demo account (Or deposit money to the real account)
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwBG3Lv_8896o0-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Download the MetaTrader 4 from the forex broker site
<p><a href="https://www.tickmill.com/trading-platforms/mt4" target="_blank" rel="noopener">Tickmill MetaTrader 4</a></p>
</li>
<li>Install the MetaTrader 4</li>
<li>Login to the account registered in MetaTrader 4
<p><img src="https://www.eightcap.com/en/wp-content/uploads/sites/11/2019/09/dl-install-login-mt4-11-edit.jpg" alt="" width="1024" height="514" /></p>
</li>
<li>Extract the Programs.zip. There will be 2 folders: Experts and Indicators. Copy these two folders.</li>
<li>Click "File" -&gt; "Open Data Folder"
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwCgpMPV608JBE-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Enter the folder "MQL4". Paste the 2 folders copied in step 7</li>
<li>Go back to MetaTrader 4. Click "File" again in the menu, click "New Chart", click "EURUSD"</li>
<li>Right-click on any Expert Advisor. Click "Refresh"
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwCplygcepPbvU-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Now the project Expert Advisor(name: 12_Tech_Indicators_EA) should appear. Drag and drop it to the EUR/USD chart</li>
<li>Go to the "Common" tab. Check the "Allow live trading" checkbox if it is unchecked. Click OK
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwA8qKunQNrZx0-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Press down the &ldquo;AutoTrading&rdquo; button:
<p><img src="https://c.mql5.com/mql4/forum/2014/03/allow_autotrading_7_1.png" alt="" width="1024" height="514" /></p>
</li>
<li>A smiley face at the right top corner of the chart means the Expert Advisor is trading automatically
<p><img src="https://img.nikonsrc.com/image/v_muduX9jHoVjeVK1lfHdGbnlmrcuA7CFcVZD_FQlwCasNxvhR1SQk-cZqK6FRukgpeLFCvWMp0/item.JPG" alt="" width="1024" height="514" /></p>
</li>
<li>Setup finished!</li>
</ol>
